-------------------------CLOUD SERVICES-------------------------------



Docker was extensively used to create different containers. Python Scripts were written based upon the needs of the user:

-STaas(Storage as a Service)

-SaaS(Software as a Service)

-PaaS(Platform as a Service)

-CaaS(Container as a Service)

A basic HTML page allows the user to choose between the options. Python-CGI at the backend runs the corresponding scripts.
